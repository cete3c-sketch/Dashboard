Chatbot

The CETE-AKU Chatbot is an AI-powered virtual assistant developed to support employees of the CETE department at Aga Khan University (AKU). The chatbot is trained using a comprehensive collection of internal documents, policies, guidelines, and departmental information, enabling it to provide accurate, consistent, and instant responses to employee queries. This project aims to streamline access to institutional knowledge, reduce response time for routine inquiries, and enhance overall employee experience within the department. By leveraging natural language processing (NLP) and machine learning, the chatbot understands user intent and delivers contextually relevant information related to CETE policies, procedures, and operations.

3CS

The Covariate Ranking App is an interactive web application designed to help users upload and organize datasets for analysis and covariate ranking. Built using Python ,React.js and styled with Tailwind CSS, it provides a smooth and responsive interface where users can securely log in and upload either single files or entire folders. The purpose of this project is to simplify the data preparation process for researchers, analysts, or machine learning practitioners by offering a user-friendly platform to manage and prepare their data before performing deeper statistical or predictive analysis.
